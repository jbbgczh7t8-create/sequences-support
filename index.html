Sequences Timer

An app for creating custom sequences with timers.

Contact Support

About

With this app you can create any series of steps within a sequence to perform any task you do regularly.

Features

• No limit on Sequences
• No limit on steps within a Sequence
• When running a sequence, each step can be announced
• Gaps of time can be made within steps.
• Different sounds can be made at the end of each step and the end of the sequence
